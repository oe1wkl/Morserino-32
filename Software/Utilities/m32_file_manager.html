How It Works
1
Connect your Morserino to this computer via USB.
2
Click Connect and select the serial port from the pop-up.
3
The file list loads automatically. You can upload or delete files.
4
Select an upload target: CW Player text, success/error sounds, or a custom path.
5
For Custom mode, edit the target path to include folders, e.g. /sounds/alert.mp3.
